#  Lojinha da Marcenaria Infantil

Site vitrine da **Lojinha da Marcenaria** — peças artesanais em madeira com foco em móveis infantis e peças únicas. Os produtos são exibidos no site e redirecionam para o **Mercado Livre** para a compra.

---

##  Sobre o Projeto

Site estático de uma página (single-page) com as seções:

- **Início** — apresentação da lojinha com chamada principal
- **Sobre Nós** — história e diferenciais da marcenaria
- **Produtos** — catálogo com filtros por categoria e link direto para o Mercado Livre
- **Contato** — formulário de mensagem + informações de contato

---

##  Tecnologias Utilizadas

| Tecnologia | Uso |
|---|---|
| HTML5 | Estrutura da página |
| CSS3 | Estilo, responsividade e animações |
| JavaScript (Vanilla) | Menu mobile, filtros de produto, formulário |
| Google Fonts | Nunito + Playfair Display |
